# I'm A Movie Wimp
A Chrome extension for people who can't handle not knowing how a movie ends. Search any movie on Google, click the button, and find out if it ends happy, sad, or ambiguous without spoilers.

##How it works:
1. Navigate to a movie's Google page (or any page with a movie title)
2. Click the I'm A Movie Wimp extension icon
3. Hit "Reveal Ending"
4. Get a one-word verdict: happy, sad, or ambiguous

The extension reads the movie title from the active tab, sends it to the Groq API, and returns a classification without spoilers

##To use this extension:
1. Download this repo
2. Go to chrome://extensions/
3. Click load unpacked and select this repo's folder as the source
